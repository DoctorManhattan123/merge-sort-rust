# Merge sort in Rust

## Requirements

* [Rust](https://www.rust-lang.org/tools/install)

If not specified, the path you execute the further commands from is the project root folder.

## Run

```sh
cargo run
```

## Build

```sh
cargo build
```


## Contribute

You can contribute shorter code or other merge sort versions.

Before contributing check that the codebase builds and run the [rust formatter]().

```sh
cargo fmt
```

If not installed:

```sh
rustup component add rustfmt
```
